The following links will take you to my github repositories that contain an agent based model and an application that will combine maps of the UK in a Jupyter notebook (under construction). For further details on these two projects please see their sections that have been detailed below.

Agent Based Model

This model was created during practical sessions to give experience of coding with Python and provide a framework that students can use when they come to code their independent project. The basic elements of what this model does are:

  • Builds agents in space
  • Provides each agent a list of the others so they can interact
  • Read in environmental data from a web file
  • Make the agents interact with the environment
  • Randomise the order of the agent actions
  • Display the model as an animation
  • Provides a GUI to run the model from

Independent project

My independent project is based off of a site location problem and will be completed in a Jupyter Notebook. It will involve reading in 3 maps of the UK that show geology, transport network and population density and combining them. Overall it will:

  • Read in 3 maps of the UK showing different factors
  • Have a GUI that allows the user to combine the maps
  • Combining/merging of the maps will be done via scrollbars
  • The 3 values for each pixel will be multiplied by the scrollbar value
  • The values for each map will then be added together
  • The numbers will be re-ranged so they are within 0 and 255
  • The user will be able to save a weighted map in a raster style CSV Text file
