ABOUT THIS SITE

This site was created by basing the CSS and some of the HTML from a template sourced from templated.co. This site provides a large collection of templates which they have very helpfully released under the creative commons (license details can be seen here). To create this site I used elements of their CSS templates and some of their HTML template to create the look I was going for. In the future I aim to completely hand code the website myself and neaten up my HTML and CSS. If you right click on the page and choose view page source you will see my code and where I have commented code that was taken from the template. If you like the style and wish to see the souce material it is known as adequately on the templated.co webpage.

Another very useful resource I used to help me code this website was w3schools.com whose website gives good and in-depth advice on HTML and CSS coding elements and syntax and provides examples for the user to try themselves. Without this website as a reference guide I would not have been able to build the site you see today, even with the use of a template. They also provide information on using other languages such as Javascript so I would recommend this site to any budding web page builders. I can say for certain that further updates to this site will be made by heavily referencing w3schools.com.

As well as the resources detailed above I also used a text editor known as Brackets. This text editor includes a real time live preview that connects to a browser window so that the user can see changes as soon as they make them. I can't put into words how much easier this made the process of building this website for me. It certainly made it less frustrating as I could instantly see if something was going wrong or wouldn't look how I wanted it to. This is another resource which I can guarantee I will be using extensively in the future, and don't think I would bother trying to build a website again without. Basically it has spoiled me! Now if it could just include a spellchecker I would be set!
